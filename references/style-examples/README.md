# Style Examples（灵感范例，非运行时依赖）

这里的 `theme-*.css` 只是**设计参考**，展示几种配色系统与标题处理手法。

- 运行时**不依赖**它们；`build_html.mjs` 用的是 Claude 现场生成的主题（`--theme-css`）。
- 用法：从中**借鉴模式**（强调色 + 中性灰阶的搭配、标题分隔方式、密度节奏），而不是挑一个直接套。
- 真正的生成规范见 `../theme-design-guide.md`。

包含范例：

- `theme-general.css` —— 通用克制，平衡百搭。
- `theme-tech.css` —— 理性蓝、紧凑、工程感。
- `theme-finance.css` —— 藏青、偏衬线、稳重。
- `theme-creative.css` —— 暖色、现代、标题左竖条。
