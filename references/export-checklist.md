# 导出检查清单

生成 HTML 预览前后、出 PDF 后，按这份过一遍。

## 内容忠实（本 skill 的硬边界）

- [ ] 简历文字与用户 Markdown 一致，**没有改写、润色、增删内容**。（内容优化是另一个 skill 的事。）
- [ ] 语言为纯中文或纯英文其一，**没有中英混排**（除非用户明确要求双语，本 skill 默认不做）。

## 排版与分页

- [ ] A4 纸张；预览里像一张居中的纸。
- [ ] 标题没有孤悬在页底（`break-after: avoid` 生效）。
- [ ] 单条经历/列表项没有被拆到两页（`break-inside: avoid` 生效）。
- [ ] 没有从一行中间截断的尴尬断页。
- [ ] 若某条经历过长导致排版紧张：**本 skill 不改写内容**，只能提示用户精简原文，或由主题调整密度。

## 视觉

- [ ] 主题是为**当前目标岗位**定制的，不是固定模板的复制。
- [ ] `--accent` 强调色统一、对比度够，打印不会糊。
- [ ] 中文字体正常（无方块/缺字）。
- [ ] 若启用了图标（`--icons` 或手动放置）：Lucide 图标显示正常、与文字对齐、颜色协调；**不用图标也完全可以**。

## PDF 输出

- [ ] 用户**已预览并确认** HTML 后才导出 PDF。
- [ ] PDF 打开后**文字可选、可复制**（矢量文字保真）。
- [ ] 缩放放大后文字边缘清晰、不糊（矢量保真）。
- [ ] 颜色/背景在 PDF 里与预览一致（`printBackground: true`）。
